## Приложение для управления цветочным магазином
Данное приложение было разработано в рамках учебной практики. Программа представляет собой комплексную систему автоматизации для цветочного магазина, позволяющую вести учёт товаров, оформлять продажи, отслеживать остатки на складе и формировать отчёты о деятельности магазина.

## Возможности приложения
- **Управление товарами** - добавление, редактирование и удаление товаров
- **Каталог продукции** - просмотр всего ассортимента с удобной навигацией
- **Корзина покупок** - формирование заказов и оформление продаж
- **История продаж** - отслеживание всех совершенных транзакций
- **Складской учёт** - контроль остатков и预警 о низком запасе товаров
- **Аналитика и отчёты** - статистика продаж, выручка, средний чек
- **Поиск товаров** - быстрый поиск по названию и категориям

## Установка

### Для пользователей
1. Скачайте последний релиз приложения из раздела Releases
2. Запустите установочный файл `FlowersShop Setup.exe`
3. Следуйте инструкциям установщика
4. После завершения установки ярлык приложения появится на рабочем столе

### Для разработчиков
1. Клонируйте репозиторий
2. Откройте решение `FlowersShop.sln` в Visual Studio 2022
3. Восстановите NuGet-пакеты
4. Соберите проект (Build → Build Solution)
5. Запустите приложение (F5)

## Требования к системе
- **Операционная система**: Windows 10/11 (64-bit)
- **.NET Framework**: 4.8 или выше
- **База данных**: SQLite (встроенная, не требует установки)
- **Дисплей**: 1280×720 или выше
- **Оперативная память**: 2 ГБ и более
- **Свободное место**: 50 МБ

## Использование

### Главный экран
- **Статистика** - отображение общей информации о товарах и продажах
- **Меню навигации** - быстрый переход к основным разделам

### Управление товарами
- **Просмотр** - весь ассортимент в удобной таблице
- **Добавление** - заполните форму с названием, категорией, ценой и количеством
- **Редактирование** - изменение существующих позиций
- **Удаление** - удаление товаров из каталога
- **Поиск** - быстрый поиск по названию

### Работа с продажами
- **Корзина** - выберите товары и укажите количество
- **Оформление** - подтверждение продажи и списание товаров со склада
- **История** - просмотр всех совершенных продаж с детализацией

### Отчёты и аналитика
- **Общая статистика** - выручка, количество продаж, средний чек
- **Контроль остатков** - товары с низким запасом
- **Экспорт данных** - выгрузка отчётов в CSV-формат
